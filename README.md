### SDLC - Software development life cycle

    Plan, Design
    Coding, Development
    Testing
    Deploy, deliver
    Monitoring, maintainance

IT company <- Flipkart (1 year)

- SDLC Follow
- Business Grow

2) Agile Model - (sprint 2-3 month)
